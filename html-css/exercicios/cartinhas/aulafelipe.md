# Estilizando:

```html
<html>
    <style>
        p {
            color: red;
        }
        .vermelho {
            color: blue;
        }
        #ver {
            color: green;
        }
    </style>
    <body>
    <p class='vermelho' id='ver'>exemplo1</p>
    <p class='vermelho'>exemplo2</p>
    <p>exemplo3</p>
    </body><!--o ex1 fica da cor verde pelo id, enquanto o ex2 fica azul por causa do class-->
</html><!--e o ex3 fica vermelho pela estilização do p em geral-->
```
