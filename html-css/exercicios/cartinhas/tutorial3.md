# Como colocar imagens no seu site:

```html
<body>
    <p>Abaixo você vai ver uma imagem que está na mesma pasta.</p>
    <img src="nome_da_imagem.png" alt="nome">
    <p>Podemos também carregar imagens que estão em sub-pastas.</p>
    <img src="nome_da_sub-pasta/nome_da_imagem.png" alt="nome">
    <p>Também podemos carregar imagens externas</p>
    <img src="nome_da_pagina_da_imagem.png" alt="nome"> <!-- URL -->
</body>
```
